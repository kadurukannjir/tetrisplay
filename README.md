# TetrisPlay

**Play Tetris free online** — a classic block puzzle game playable directly in your browser with no downloads or sign-ups required.

## Features
- 🎮 Full Tetris gameplay (7 tetromino types, ghost piece, level progression)
- 📱 Mobile-friendly with touch controls
- 📝 Gaming blog with tips, history & science articles
- 🔐 Privacy Policy, Terms of Service, Disclaimer
- ⚡ Built with React + Vite + TypeScript + Tailwind CSS

## Live Demo
[Play at https://kadurukannjir.github.io/tetrisplay/](https://kadurukannjir.github.io/tetrisplay/)

## Tech Stack
- React 18 + TypeScript
- Vite 6
- Tailwind CSS v4
- Wouter (routing)
- TanStack Query
